How to remove a Header

To remove a header from your files, enter the exact header text you want to delete into the Enter Header field.

This ensures that the specified header will be identified and removed from the selected files.

Make sure to select the appropriate file types to apply the changes effectively.

How to use HeaderMaker

Welcome to HeaderMaker, your go-to tool for adding custom headers to your files with ease. Follow these steps to get started:

  1. Select Project Folder: Click on the Select Folder button to open the main project directory where you want to add headers. This step ensures that the headers are applied to the correct files within your project.
  2. Enter the Header: Type the custom header text you want to add in the Enter Header field.
  3. Select a Template: Click on the Select Template button to choose a predefined header.
  4. Select File Types: Choose the file types to which you want the header added. Ensure you select all the relevant types to apply the changes effectively.
  5. Preview Changes: Use the preview feature to see how the headers will look in your files before finalizing the changes.
  6. Apply Changes: Once you’re satisfied with your header and selections, click Add Headers to apply the headers to the chosen files.

By following these steps, you can efficiently manage and update file headers, ensuring consistency and professionalism across your projects.
